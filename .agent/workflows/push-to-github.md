# Push Changes to GitHub

Whenever you make changes to your portfolio files, follow these steps to update GitHub:

## Quick Push Workflow

// turbo-all

1. **Check what files changed**

```bash
git status
```

1. **Stage all changes**

```bash
git add .
```

1. **Commit with a message**

```bash
git commit -m "Your description of changes here"
```

1. **Push to GitHub**

```bash
git push
```

## Examples of Good Commit Messages

- `git commit -m "Updated homepage hero section"`
- `git commit -m "Added new case study page"`
- `git commit -m "Fixed contact form styling"`
- `git commit -m "Updated about page content"`
